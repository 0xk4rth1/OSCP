
List all available file shares exported by the target:

`showmount -e <ip>`

List any connected remote clients and the shares they are accessing

`showmount -a <ip>`

Enumerating `nfs` using `nmap`

`nmap -p 2049 --script=nfs-showmount,nfs-ls,nfs-statfs <TARGET-IP>`

**Mounting the Share locally:**  (Attacker machine)

`mkdir /tmp/mount`        (create a folder for mounting point)

`sudo mount -t nfs <target-ip>:/var /tmp/mount`      (var directory will be mounted to /tmp/mount)
