
To connect smb with default credentials (Guest account)

`crackmapexec smb <ip> -u Guest -p "" `   (it will confirm whether the account is active or not)

Alternatively, we can use `nxe`

`nxe smb <ip> -U Guest -p ""`

Additional parameters:

`--shares`  (for listing shares)
`--users`    (it'll list users)

To connect using `smbclient`

`smbclient //<ip>/<share_name> -U Guest`
